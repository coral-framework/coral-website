Methods

bool canLoadModulePart(in string moduleName)

Returns whether a module part can be loaded from moduleName.

This checks if the repository contains a loadable module artifact in namespace moduleName. For example, the default loader would check if a dynamic library file named moduleName exists in moduleName.

co.IModulePart loadModulePart(in string moduleName)

Loads a module part from namespace moduleName.

Any exception raised by this method will force the module’s abortion.
