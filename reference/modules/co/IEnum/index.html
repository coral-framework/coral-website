co.IEnuminterface

Describes an enum type.

Contains a list of identifiers with implicit values from 0 to n-1.

Methods

int32 getValueOf(in string id)

Finds an identifier and returns its value.

Returns -1 if id is not a valid identifier. Valid results are in the range [0, n-1].
