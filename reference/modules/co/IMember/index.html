co.IMemberinterface

Interface for members of a co.ICompositeType.

Each type member has a unique name within its owner’s hierarchy.

  • Tags:
    • Type Introspection
  • Referenced from 4 types: co.ICompositeType, co.IField, co.IMethod, co.IPort.

Hierarchy

  • co.IServiceBase of all interfaces. Represents a service provided by an object.
    • co.IAnnotatedBase interface for services that can be annotated.
      • co.IMemberInterface for members of a co.ICompositeType.
        • 3 known subtypes: co.IField, co.IMethod, co.IPort.
